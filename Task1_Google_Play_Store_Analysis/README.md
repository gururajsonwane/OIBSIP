# Google Play Store Analysis

## Project Overview
This project focuses on analyzing Google Play Store applications using Python libraries such as Pandas, Matplotlib, and Seaborn.

The analysis includes:
- Data cleaning
- Data visualization
- Insights extraction
- Understanding app trends and user behavior

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Information
The dataset contains information about:
- App categories
- Ratings
- Reviews
- Installs
- Pricing
- User reviews

---

## Key Insights
- Most applications on the Play Store are free apps.
- Family category contains the highest number of applications.
- Most app ratings are between 4.0 and 4.5.
- Categories such as GAME and COMMUNICATION have the highest installs.
