# Google Play Store Analysis

## About the Project
This project was developed as part of my Oasis Infobyte Data Analytics Internship.

The main objective of this project is to analyze Google Play Store application data and extract meaningful insights using Python. The project includes data cleaning, visualization, and analysis of app categories, ratings, installs, and pricing trends.

---

## Objectives
- Perform data cleaning and preprocessing
- Analyze app categories and installs
- Visualize important patterns using charts and graphs
- Extract useful business insights from the dataset

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Information
The dataset contains details related to:
- Application categories
- User ratings
- Reviews
- Number of installs
- Pricing
- User feedback

---

## Visualizations Included
- Top App Categories
- Free vs Paid Apps Distribution
- Ratings Distribution
- Most Installed App Categories

---

## Key Insights
- Most applications on the Google Play Store are free apps.
- The Family category contains the highest number of applications.
- Most app ratings fall between 4.0 and 4.5.
- GAME and COMMUNICATION categories have the highest installs.
