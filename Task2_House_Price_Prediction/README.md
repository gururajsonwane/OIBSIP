# House Price Prediction using Linear Regression

## About the Project
This project was developed as part of my Oasis Infobyte Data Analytics Internship.

The main objective of this project is to predict house prices using Linear Regression based on important house features such as area, bedrooms, bathrooms, stories, and parking.

---

## Objectives
- Perform data cleaning and preprocessing
- Select important house features
- Build a Linear Regression model
- Predict house prices
- Evaluate model performance
- Visualize actual and predicted prices

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Jupyter Notebook

---

## Dataset Information
The dataset contains information related to:
- House prices
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Furnishing status

---

## Machine Learning Workflow
- Data Loading
- Data Cleaning
- Feature Selection
- Train-Test Split
- Linear Regression Model
- Prediction
- Model Evaluation
- Data Visualization

---

## Model Evaluation
- Mean Squared Error (MSE)
- R2 Score

The model achieved a moderate R2 Score, showing reasonable prediction capability.
